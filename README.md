# 🐍 PyFetch

[![Python](https://img.shields.io)](https://www.python.org)
[![License: MIT](https://img.shields.io)](https://opensource.org)

**pyfetch** — это быстрый и настраиваемый инструмент для вывода информации 

"Как neofetch, только на змеином языке" 🐍

✨ Особенности

    🚀 Скорость: Минимальное время запуска.
    🎨 Кастомизация: Легко менять ASCII-арт и цвета через конфиг.
    🐍 Pure Python: Никаких тяжелых зависимостей
    💻 Кроссплатформенность: Работает везде, где есть Python.


## 🎨 ASCII Art Status

В текущей версии реализована эксклюзивная поддержка логотипа для:
- [x] **CachyOS** 🐧
- [x] Arch Linux (WIP)
- [ ] Fedora (WIP)
- [ ] Debian (WIP)

> [!TIP]
> Если вы запустите `pyfetch` на другом дистрибутиве, информация о системе отобразится корректно, но вместо логотипа вы увидите стандартную змейку Python.
